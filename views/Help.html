Notes Maker and Organizer

This Application having Add,Edit,Delete,Search the Notes and its description for students.
In main menu that contain 5 Option Add,Update,delete,Add remainder,Lock application.
The add Option which is used to add a note,key points,description to storage.
The Edit Option which is used to Edit/update a note,key points,description to storage.
The Delete Option which is used to delete entire note from storage.
The Add remainder which is used to add remainder and yet to be study section.
Lock Option which is used to lock the application with password protected.
